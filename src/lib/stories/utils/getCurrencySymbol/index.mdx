# getCurrencySymbol

This utility function retrieves the standard **currency symbol** associated with a 3-letter alphabetic currency code (ISO 4217). It's a quick and reliable way to get symbols like '$', '€', or '¥' using the browser's built-in internationalization API (`Intl.NumberFormat`).

<Source
  dark
  language="ts"
  code={`
import { getCurrencySymbol } '@flightlesslabs/number-format';

// Usage
getCurrencySymbol('USD') // '$'
getCurrencySymbol('EUR') // '€'
getCurrencySymbol('GBP') // '£'
`}
/>
